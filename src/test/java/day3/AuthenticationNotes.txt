Oauth2.0
--------------

1) From the application (Manual process)
	1) Client ID
	2) Client Secrete

2) Send Post request for getting token
	POST https://api.imgur.com/oauth2/token
		ClientID
		Client secrete
		tokenURL
		Redirect URL
		Grant type
		Authorization code

you will get token once POST request is succesfull.

3) Use Token to do API call ( Get request).
